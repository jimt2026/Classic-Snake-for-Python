# Classic-Snake-for-Python
# Version 1.1
# 24/04/2026

You will need Pygame to play this. 

You should see a terminal window and a smaller, 1:1 size window with the game on it. 
Use WASD to move the snake. 
Press R to restart, or Q to quit when you lose.
You lose by either hitting the edges or hitting yourself. 
Eat the red pixels (Food) to grow the snake. 
Your score is in the top left corner. 

My high score is 42, can you beat it?
